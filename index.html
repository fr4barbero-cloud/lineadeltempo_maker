GUIDA RAPIDA

Tre passaggi per creare la linea del tempo

Apri la guida solo quando ti serve. Puoi iniziare subito anche senza leggerla.

1IniziaScegli se creare una nuova linea del tempo o modificare una cartella già creata.
2Crea e controllaInserisci le card una alla volta. In ogni card puoi mettere un’immagine sul fronte, un testo sul retro, il file audio MP3 del testo e uno o più collegamenti esterni. Per le immagini usa solo file PNG. Controlla il risultato con l’anteprima.
3Scegli il risultatoAlla fine scarica il progetto completo se vuoi riaprirlo e modificarlo; scarica invece il progetto da pubblicare o consultare se vuoi condividerlo senza permettere modifiche. Nella seconda sezione trovi anche i tutorial per pubblicare su GitHub.

Hai bisogno di altre istruzioni?

Scegli soltanto l’aiuto che ti serve.

Come riaprire una linea del tempo

  1. Premi “Apri una linea del tempo”.
  2. Seleziona la cartella completa, non il solo file index.html.
  3. Modifica, aggiungi, elimina o riordina le card.
  4. Alla fine premi “Scarica il progetto” per conservare una nuova copia modificabile.

Importante: la cartella deve contenere progetto.timeline.

Scarica il progetto

Scarica una copia che potrai riaprire e modificare.

Scaricare e continuare a modificare

  1. Scegli una cartella vuota.
  2. Il programma scarica tutti i file del progetto nella cartella scelta.
  3. Conserva la cartella completa senza separare i file.
  4. Per continuare il lavoro, usa “Apri una linea del tempo”.

La cartella contiene progetto.timeline, necessario per riaprire il progetto nell’editor.

Scarica il progetto da pubblicare o consultare

Scarica una copia che potrai condividere senza permettere modifiche. Poi segui il tutorial per pubblicare la tua linea del tempo online e ottenere un link.

Consigliato per la condivisione

Come funziona
  1. Scegli una cartella diversa da quella del progetto modificabile.
  2. Il programma crea una copia che non può essere modificata con l’editor.
  3. Per consultarla sul computer, apri il file index.html.
  4. Per pubblicarla online e ottenere un link, segui uno dei tutorial qui sotto.

Questa copia non contiene il file progetto.timeline.

Pubblicare la linea del tempo online

Segui il tutorial che preferisci per pubblicare la tua linea del tempo su GitHub e ottenere un link da condividere.

Pubblicare online e ottenere un link

Prima crea la copia di sola consultazione. Poi carica tutti i file della cartella su GitHub Pages.

  1. Accedi a GitHub e crea un nuovo repository pubblico.
  2. Apri la cartella della copia di sola consultazione.
  3. Carica tutti i file e tutte le cartelle che contiene.
  4. Conferma il caricamento con Commit changes.
  5. Apri Settings → Pages.
  6. Scegli Deploy from a branch, poi main e /(root).
  7. Salva e apri il link mostrato da GitHub.

Non caricare soltanto index.html: servono tutti i file della cartella.
